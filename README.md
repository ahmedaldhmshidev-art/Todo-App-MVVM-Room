# Todo App 

Todo App هو تطبيق لإدارة المهام اليومية بطريقة بسيطة وفعّالة. يسمح للمستخدم بإضافة مهام جديدة، تعديلها، وحذفها بسهولة مع واجهة جذابة وتجربة مستخدم سلسة.

الميزات الرئيسية:
إضافة مهمة جديدة باستخدام Bottom Sheet Fragment.
تحديد اسم المهمة، وصف مختصر، ووقت المهمة.
حفظ المهام في قاعدة بيانات Room.
عرض المهام في RecyclerView.
تعديل مهمة موجودة عند الضغط عليها.
حذف مهمة بالضغط المطول.
وضع علامة صح على المهام المكتملة: عند الانتهاء من المهمة، يمكن تفعيل علامة ✔️، لتظهر المهمة مع خط فوق الاسم والوقت.
استخدام ViewBinding لتسهيل التعامل مع واجهة المستخدم.
تدفق البيانات باستخدام Flow وتحويلها إلى LiveData في ViewModel لمزامنة البيانات بشكل مباشر مع واجهة المستخدم.

التقنيات المستخدمة:
Kotlin كلغة برمجة.
Android Jetpack MVVM لتصميم هيكل التطبيق.
Room Database لتخزين البيانات محليًا.
Repository & Factory Pattern لإدارة البيانات بشكل منظم.
Coroutines (suspend functions) لمعالجة العمليات غير المتزامنة.
Flow & LiveData لمزامنة وعرض البيانات الحديثة في واجهة المستخدم.
RecyclerView Adapter لعرض قائمة المهام.
ViewBinding لتبسيط التعامل مع عناصر واجهة المستخدم.

كيفية الاستخدام:
افتح التطبيق واضغط على زر إضافة مهمة.
أدخل اسم المهمة، وصفها، وحدد الوقت.
اضغط على حفظ لتظهر المهمة في القائمة.
لتعديل المهمة، اضغط عليها لمشاهدة واجهة التعديل.
لحذف مهمة، اضغط مطولًا عليها.
لإنهاء المهمة، اضغط على علامة ✔️ لتطبيق خط فوق اسم المهمة ووقت تنفيذها.
